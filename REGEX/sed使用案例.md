```txt

BAI王璐瑶,改签待处理
BAI王璐瑶,业务统计

BAI王童,电话记录
BAI王童,改签待处理
BAI王童,在线及短信回复模板配置
```

已按照用户名排序，现需按照用户名，将后面的权限拼接起来。

vim不好搞，用 sed + vim

先用

```shell
sed -r ':a;N;s/(.*),(.*)\n(\1),(.*)/\1,\2+\4/g;ta'
```

得到如下

BAI王璐瑶,账单下载+改签待处理+业务统计+改签订单 \
BAI王童,订单管理 \
BAI王童,已登记退票报表+我的历史事件+Qunar退款完成订单+快速出票订单+电话记录

下一步用 vim

```shell
 %s/\(.*\),\(.*\)\n\(\1\),\(.*\)/\1,\2+\4/g
```

将剩下分离的行再次合并。
